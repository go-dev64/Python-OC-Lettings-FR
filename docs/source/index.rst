Welcome to OC_lettings_site's documentation!
============================================

Orange County Lettings is a property rental company. On the site, you can view several rental locations as well as user profiles.


Used technology
===============

The **Oc_lettings_site** is powered by:

   - **Python 3.10** - `documentation Python <https://docs.python.org/3.10/>`_.
   - **Django 3.0** - `documentation Django <https://docs.djangoproject.com/en/3.0/>`_.
   - **Sqlite3** Database
   - Versionning by **GITHUB**
   - Monitoring by **Sentry** - `documentation Sentry <https://docs.sentry.io/>`_.
   - Contenairisation by **Docker** - `documentation Docker <https://docs.docker.com/>`_.
   - CI/CD by **Cercle CI** -  `documentation CircleCI <https://circleci.com/docs/>`_.
   - Hosting by **Render** -  `documentation Render <https://render.com/docs>`_.

   This project is a school project.
